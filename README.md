# COVID Analysis Dashboard – Power BI Project

## 📌 Project Overview
This project is an interactive COVID-19 Analysis Dashboard developed in Power BI using world COVID statistics data.
The dashboard provides insights into total cases, deaths, recoveries, testing statistics, population distribution, and continent-wise analysis through interactive visuals, slicers, drill-through pages, and bookmarks.

The project was created as part of a dashboard development assignment with specific UI/UX and visualization requirements.

## 📊 Dashboard Pages
### 1️⃣ Index Page
The Index page acts as the navigation hub for the report.

**Features**
- Interactive navigation buttons
- Direct page switching using bookmarks
- Professional dark-themed UI design

**Navigation Pages**
- Index
- Covid Analysis
- Bookmark
### 2️⃣ COVID Analysis Page
This page contains the core analytical dashboard with multiple KPIs and interactive visualizations.

**KPI Cards**

The dashboard includes the following KPI cards:
- Total Tests
- Total Cases
- Total Deaths
- Total Recovered
- Active Cases

**Interactive Elements**
- Country/Region slicer
- Continent slicer
- Back button navigation
- Drill-through functionality

**Visualizations Used**
- *Pie Chart →* Total Cases, Total Deaths, Total Recovered
- *Column Chart →* Total Cases by Continent with drill-down hierarchy
- *Combo Chart →* Active Cases & Total Tests by Country/Region
- Top 10 Countries based on Highest Death Count

**Drill Down Hierarchy**
- Continent
- Country/Region

**Drill Through Feature**

Users can right-click a country from the combo chart and drill through to the Bookmark page for detailed country-level insights.

### 3️⃣ Bookmark Page
This page is designed for detailed country-level analysis.

**Table Visual Includes**
- Country/Region
- Population
- Total Tests
- Total Cases
- Total Recovered
- Total Deaths

**Additional Features**
- Population bar chart by Country/Region
- Bookmark toggle button
- Dynamic Data Labels ON/OFF functionality
The bookmark functionality allows users to switch between:
- Chart with data labels
- Chart without data labels
